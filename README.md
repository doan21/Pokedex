
# Pokédex App 🎮

A command-line Pokédex built in Python that lets you search and explore Pokémon stats from a CSV database. Includes a surprise Pokémon GO-style game hidden inside!

## Features

- 📋 Display a selected number of Pokémon with full stats
- 🔍 Search Pokémon by type
- 📊 Find Pokémon by total base stat
- ⚔️ Filter Pokémon by minimum stats (HP, Attack, Defense, etc.)
- 🌟 Display all legendary Pokémon by type
- ⚡ 1v1 Pokémon battle mode with type advantages
- 🕹️ Secret Pokémon GO-style exploration game (try option 7!)

## How to Run

Make sure you have Python installed, then:

```bash
python app.py
```

Make sure `Pokemon.csv` is in the same folder as `app.py`.
